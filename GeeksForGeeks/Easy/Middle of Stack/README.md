<h2><a href="https://www.geeksforgeeks.org/problems/middle-of-stack/1?page=5&category=Stack&sortBy=submissions">Middle of Stack</a></h2><h3>Difficulty Level: Easy</h3><hr><p><span style="font-size: 18px;">Given a stack<strong> st[] </strong>of size <strong>n</strong>, find the middle element of the stack</span></p>
<p><span style="font-size: 18px;"><strong>Note:</strong> In case of even elements, return the middle at the bottom.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> st[] = [1, 2, 3, 4, 5]
<strong>Output:</strong> 3
<strong>Explanation:</strong> The middle elements is 3.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> st[] = [1, 2, 3, 4]
<strong>Output:</strong> 2
<strong>Explanation:</strong> The two middle elements are 2 and 3. Since we need the middle element closer to the bottom, the answer is 2.</span></pre>
<p><span style="font-size: 18px;"><strong>Constraints:</strong><br>1 ≤ st.size() ≤ 10<sup>5</sup></span></p>
